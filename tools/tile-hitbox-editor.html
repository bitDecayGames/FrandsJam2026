Tile Hitbox Editor - Help

Loading
1. Click "Load .ldtk" and select your LDTK project file
2. Click "Load Tileset Image" and select the tileset PNG
Drawing Polygons
Click on the editor to place vertices (snapped to pixel coords)
Double-click or right-click near the first vertex to close the polygon
Drag vertices to reposition them
Shift+click a vertex to delete it
Presets
Use preset buttons for common shapes (full, halves, corner triangles)
Navigation
Click tiles in the grid, or use Prev/Next buttons
Arrow keys also navigate (Left/Right, Up/Down)
"Tagged only" filter shows only SOLID/SHALLOW tiles
Copy/Paste
Copy polygon from current tile, select another tile, then Paste
Mirror H/V flips the current polygon horizontally/vertically
Undo
Removes the last placed vertex (Ctrl+Z)
Export/Import
Export saves tile-hitboxes.json with all polygon data
Import loads a previously exported JSON to resume editing
Autosave
Data is saved to localStorage automatically
